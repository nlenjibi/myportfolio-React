# Django + React Portfolio Backend (Dockerized)

This is a production-ready, Dockerized full-stack portfolio system using Django (backend) and React (frontend).

## Features
- Django REST Framework backend with JWT authentication
- React frontend consuming the API
- PostgreSQL database
- Dockerized for easy deployment
- Admin interface for content management

## Structure
- Django backend with modular apps for each portfolio section
- React frontend consuming REST APIs
- Docker Compose for orchestration
- JWT-based authentication system

## Requirements
- Docker
- Docker Compose
- Node.js (for development)
- Python 3.8+
